# 04_采购销售与消费者

## 这个类别是干什么的

服务于采购合同、销售合同、交付验收、产品质量、消费者权益、退款售后和食品相关经营风险。

## 当前法律文件

| 文件 | 文件名日期 | 当前用途 | 关联方向 |
|---|---:|---|---|
| `中华人民共和国消费者权益保护法_20131025.docx` | 2013-10-25 | 消费者权益、经营者义务、退款赔偿、格式条款风险 | 电商经营、争议解决 |
| `中华人民共和国反食品浪费法_20210429.docx` | 2021-04-29 | 餐饮、食品销售、经营提示和宣传合规 | 行业专项、市场营销 |
| `中华人民共和国产品质量法_20181229.docx` | 2018-12-29 | 产品质量责任、销售者责任、质量瑕疵和处罚风险 | 合同通用、行业专项 |
| `中华人民共和国价格法_19971229.docx` | 1997-12-29 | 价格行为、明码标价、价格欺诈和价格监管 | 电子商务、行业专项 |
