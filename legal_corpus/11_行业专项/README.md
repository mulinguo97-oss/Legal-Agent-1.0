# 11_行业专项

## 这个类别是干什么的

存放食品、危化品、危险废物等行业专项法律法规。第一版 MVP 可作为扩展知识，不建议作为默认高频召回内容。

## 建议后续补充

- 具体行业的许可管理办法、行政处罚裁量规则、监管部门问答
- 餐饮、电商食品销售、化工贸易等专项风险模板

## RAG 使用建议

- 只有用户问题明显涉及食品、化工、环保、危废时才召回本目录。
- 行业监管问题通常存在地方差异，回答时应提示核验当地监管要求。
