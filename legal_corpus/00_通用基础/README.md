# 00_通用基础

## 这个类别是干什么的

存放全项目都会引用的基础法律。合同审查、租赁、买卖、服务合作、侵权责任、人格权、争议解决等方向都会依赖这里的内容。

## RAG 使用建议

- 民法典建议按“编 / 章 / 节 / 条”切片，不要整章超长切片。
- 合同审查优先召回合同编相关条文，租赁场景召回租赁合同和物权相关条文。
- 对用户输出时，优先引用具体条文编号，不要只引用法律名称。

## 更新规则

- 如果新增民法典司法解释，不放在这里全文重复，可放入对应业务方向或 `09_争议解决`，并在本索引补充引用说明。
- 文件名日期仅代表当前文件标注日期，正式入库前需要核验公布日期、施行日期和现行有效状态。
