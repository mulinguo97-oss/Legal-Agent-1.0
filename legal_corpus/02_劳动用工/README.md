# 02_劳动用工

## 这个类别是干什么的

服务于劳动合同审查、招聘用工、试用期、工资社保、竞业限制、解除劳动关系和劳动争议提示。

## 当前法律文件

| 文件 | 文件名日期 | 当前用途 | 关联方向 |
|---|---:|---|---|
| `中华人民共和国劳动法_20181229.docx` | 2018-12-29 | 劳动关系、劳动保护、工资工时、社会保险和劳动争议基础规则 | 合同通用、争议解决 |
| `中华人民共和国劳动合同法_20121228.docx` | 2012-12-28 | 劳动合同订立、履行、解除、终止、经济补偿、竞业限制等 | 合同通用、争议解决 |
| `中华人民共和国劳动合同法实施条例.docx` | 未标注 | 劳动合同法配套规则、劳动合同订立履行解除的细化要求 | 合同通用、争议解决 |
| `中华人民共和国劳动争议调解仲裁法_20071229.docx` | 2007-12-29 | 劳动争议调解、仲裁申请、仲裁程序 | 争议解决 |
| `中华人民共和国社会保险法_20181229.docx` | 2018-12-29 | 社保缴纳、养老、医疗、工伤、失业、生育保险基础规则 | 创业公司、争议解决 |

## RAG 使用建议

- 劳动合同审查优先切出：试用期、服务期、竞业限制、解除终止、经济补偿、劳务派遣、非全日制用工。
- 涉及解除、赔偿、仲裁时，应联动 `09_争议解决`。
