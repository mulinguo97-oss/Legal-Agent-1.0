# 法律知识库分类索引

本目录按“项目方向/业务场景”组织法律资料，服务于法律小助手的合同审查、合同问答、创业经营法律风险问答和后续 RAG 检索。

## 分类原则

- 每份原始法律文件只保存在一个主归属目录，避免同一法律多处复制导致版本混乱。
- 跨场景使用的法律，在各方向 `README.md` 中通过“关联方向”说明，不重复存放全文。
- 文件名中的日期暂按“来源文件标注日期”记录，通常可能是公布、修订或版本日期；生产使用前应以国家法律法规数据库等权威来源核验。
- 后续新增新版法律时，优先保留新旧版本并在索引中标注时效状态，不直接覆盖旧文件。

## 目录说明

| 目录 | 用途 |
|---|---|
| `_incoming_new_downloads` | 新下载文件暂存区。未分类、未核验、未入库的文件先放这里。 |
| `00_通用基础` | 宪法、民法典等全局基础法律，供多个方向共同引用。 |
| `01_合同通用` | 合同成立、效力、履行、违约、解除、管辖等通用审查规则。当前主要引用 `00_通用基础` 中的民法典。 |
| `02_劳动用工` | 劳动合同、用工关系、工资社保、竞业限制、劳动争议等。 |
| `03_租赁与不动产` | 房屋/商铺租赁、交付、维修、转租、押金等。当前主要引用民法典。 |
| `04_采购销售与消费者` | 买卖、采购、销售、消费者权益、食品浪费等经营场景。 |
| `05_服务合作与电子交易` | 服务合作、电子合同、电子商务、物流寄递等。 |
| `06_创业公司与税务` | 公司设立、组织治理、股权、发票税务等。 |
| `07_知识产权与市场竞争` | 广告宣传、不正当竞争、品牌素材和商业竞争风险。 |
| `08_数据与经营合规` | 网络安全、未成年人保护、线上服务合规等。 |
| `09_争议解决` | 民事诉讼、仲裁、行政诉讼、海事诉讼等救济路径。 |
| `10_高风险兜底` | 刑事、治安、安全生产等高风险提示。 |
| `11_行业专项` | 食品、危险化学品、危险废物等行业专项监管。 |
| `99_重复待清理` | 重复文件或暂不入库文件，确认后再处理。 |
