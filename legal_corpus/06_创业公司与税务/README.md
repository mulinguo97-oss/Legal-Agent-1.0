# 06_创业公司与税务

## 这个类别是干什么的

服务于公司设立、股权合作、公司治理、印章证照、发票税务、经营主体责任等创业经营问答。

## RAG 使用建议

- 创业问答应区分“公司治理风险”“合同交易风险”“税务发票风险”。
- 涉及股权争议、虚开发票、逃税等高风险问题时，应联动 `09_争议解决` 和 `10_高风险兜底`。
