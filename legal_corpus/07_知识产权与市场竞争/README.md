# 07_知识产权与市场竞争

## 这个类别是干什么的

服务于广告宣传、商业竞争、商标字号、图片字体代码素材、商业秘密和不正当竞争风险提示。

## RAG 使用建议

- 对“图片、字体、代码、商标能不能用”的问题，应优先召回著作权法、商标法、专利法和反不正当竞争法。
- 软件外包、代码授权等问题应同时召回著作权法和计算机软件保护条例。
