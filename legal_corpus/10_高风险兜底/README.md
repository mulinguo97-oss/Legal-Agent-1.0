# 10_高风险兜底

## 这个类别是干什么的

服务于刑事、治安、安全生产等高风险识别。该目录主要用于提示“可能需要尽快咨询律师/专业机构”，不建议模型直接给确定性结论。

- 输出时避免“构成犯罪”“一定违法”等确定性表述。
- 建议用“可能涉及刑事/行政风险，需结合事实和证据判断，并建议尽快咨询律师”。
