# 新下载法律文件暂存区

这个目录用于存放以后新下载、尚未分类、尚未核验、尚未入库的法律法规、司法解释、案例或数据集文件。

## 使用规则

- 新下载的文件先放到这里，不直接放进正式方向目录。
- 文件名尽量保留来源原名；如果需要重命名，建议格式为：`法律名称_YYYYMMDD_来源.docx`。
- 不确定日期含义时，不要自行改成“施行日期”或“现行日期”，先保留下载文件中的日期。
- 文件进入正式目录前，应先确认：
  - 是否已有同名或同内容文件；
  - 是否来自权威或可信来源；
  - 是否现行有效；
  - 应归入哪个主方向；
  - 是否还需要在其他方向的 `README.md` 中做交叉引用。

## 处理流程

1. 下载新文件到本目录。
2. 比对 `legal_corpus/` 现有文件，检查是否重复。
3. 核验来源、公布日期、施行日期、修订日期和时效状态。
4. 移动到一个正式方向目录。
5. 更新对应方向的 `README.md`。
6. 如涉及多个方向，只在主目录保存原文件，其他方向通过索引引用。

## 默认状态

本目录中的文件默认不进入 RAG，不参与切片、向量化或问答检索。
